# py-askcli 0.2.0

## exemple

```python
from askcli.menu import Menu
m = Menu(['action 1','action 2', 'action 3'])
m.launch()
print('_______')
print("vous avez selectione le menu {0} : {1}".format(m.choose, m.get_choosen_text()))
```
**resultat**

```

  2 - action 2
  3 - action 2
```
